المخزن الدائري ( circular buffer ) : هو بنية تُستخدم لتخزين البيانات داخل مصفوفة ثابتة الحجم ولكن بجعل المؤشرات تعود إلى البداية عند الوصول إلى النهاية مما يتم باستخدام الذاكرة بكفاءة عالية
فكرة الكود هي ان يقوم المستخدم بإدخال:
- حجم المخزن الدائري  
- اسمه الشخصي  
ثم يقوم البرنامج بإضافة السلسلة "CE-ESY" إلى الاسم وتخزين كل حرف داخل المخزن ثم قراءة البيانات مرة أخرى وعرضها على الشاشة
